# Roastcraft Coffee Website ☕

## 📌 Project Overview
Roastcraft is a multi-page coffee e-commerce website designed to simulate a real-world online coffee store. The project focuses on user experience, product browsing, cart functionality, authentication, and a complete frontend flow of an e-commerce system.
## 🌐 Live Demo
https://sheryas444777-wq.github.io/roastcraft-website/

---

## 🚀 Features
- Multi-page website (Home, Coffee, Equipment, Events, Cart, Offers)
- Product listing with search functionality
- Shopping cart system (add/remove items)
- User authentication (Firebase + Local Storage fallback)
- Event registration system
- Payment page UI simulation
- Responsive and modern UI design

---

## 🛠️ Tech Stack
- HTML5  
- CSS3  
- JavaScript (Vanilla)  
- Firebase Authentication  
- Firebase Realtime Database

## 📸 Note on Images
Images are not included in this repository to keep the project lightweight and avoid large file uploads.

All image paths are structured inside the `/images` folder. You can add your own images (e.g., coffee beans, equipment, banners) to restore full visual appearance.
